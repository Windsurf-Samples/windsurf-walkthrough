# Extension Guidance

Windsurf, using VS Code's interface and AI, is easy to adopt for developers from VS, Eclipse, or VS Code. It uses the Open VSX Registry for extensions, accessible via the Extensions panel or website. To help you get the most out of Windsurf for different programming languages, weâ€™ve compiled a list of popular, community-recommended extensions from the Open VSX marketplace that other users have found helpful for replicating familiar IDE experiences.

### Java

| Extension | Description |
|---|---|

| [Gradle](https://open-vsx.org/extension/vscjava/vscode-gradle) | Gradle support: task explorer, project insights, and CLI integration.      |
| [Java Project Manager](https://open-vsx.org/extension/vscjava/vscode-java-dependency) | Visualize and manage Java project dependencies.                            |
